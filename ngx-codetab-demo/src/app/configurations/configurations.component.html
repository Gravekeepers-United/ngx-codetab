Install and Add PrismJs

This package requires a peer dependency of prismJs this can be installed with the command:


            {{ prismInstall }}

Once PrismJs is installed you need to update your projects angular.json to include one of the prism themes: 


            {{ prismStyles }}

NOTE: Any style can be used but we've tailed tabs content background and stuff to the tomorrow theme. Click to find more about PrismJs.
